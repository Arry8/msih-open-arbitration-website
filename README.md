# MSIH Open Arbitration Database Website

A modern, responsive website for the MSIH Open Arbitration Database project, allowing consumers to document arbitration experiences and access aggregated data about corporate dispute patterns.

## Project Overview

The Open Arbitration Database is a comprehensive platform that allows consumers to:
- Document their arbitration experiences
- Share case details (anonymously)
- Access aggregated data about corporate dispute patterns
- Build collective knowledge to strengthen future cases

## Features

- **Modern Responsive Design** - Works on all devices
- **Case Submission Form** - Anonymous contribution system
- **Database Search** - Explore arbitration cases
- **Resource Library** - Guides, templates, and FAQs
- **Analytics Dashboard** - Data visualization (planned)

## Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** Custom CSS with CSS Grid & Flexbox
- **Icons:** Font Awesome 6
- **Fonts:** Google Fonts (Open Sans, Roboto Slab)
- **Hosting:** GitHub Pages (initial)

## Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/msih-open-arbitration-website.git
   cd msih-open-arbitration-website
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # Node.js
   npx serve .
   ```

3. Visit `http://localhost:8000` in your browser.

### GitHub Pages Deployment

This site is configured for GitHub Pages. To deploy:

1. Push to the `main` branch
2. GitHub Pages will automatically deploy from the root directory
3. Site will be available at: `https://yourusername.github.io/msih-open-arbitration-website`
