# [TASK] Implement responsive mobile menu

## Task Description
The current navigation works on desktop but needs a proper mobile hamburger menu for smaller screens.

## Acceptance Criteria
- [ ] Hamburger menu icon appears on mobile screens (< 768px)
- [ ] Menu slides in/out smoothly
- [ ] Menu includes all navigation links
- [ ] Menu closes when clicking outside or on a link
- [ ] Accessible (keyboard navigation, ARIA labels)

## Technical Details
- Use CSS for animations (transforms/transitions)
- Use JavaScript for toggling functionality
- Implement touch-friendly interactions

## Files to Modify
- `index.html` (add mobile menu structure)
- `css/style.css` (add mobile menu styles)
- `js/script.js` (add mobile menu functionality)

## Dependencies
None - can be implemented independently

## Testing Instructions
1. Resize browser to mobile width
2. Click hamburger icon
3. Verify menu appears
4. Click links to navigate
5. Click outside to close
6. Test on actual mobile devices

- [x] Small (< 1 hour)
